The parallel write algorithm is 

```
for i from 0 to nlen:
    set address <address + i>
    set data <nbytes[i]>
    delay for <ADDRESS_HOLD_TIME>
    enable chip 
    delay for <PULSE_WIDTH_TIME>
    disable chip
```

The parallel read algorithm is 

```
enable chip 
for i from 0 to nlen
    set address <address + i>
    delay <ADDRESS_HOLD_TIME>
    nbytes[i] = get data
disable chip
```
